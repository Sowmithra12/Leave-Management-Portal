# Employee Leave Management Portal

A full-stack Leave Management Portal built using ASP.NET Core Web API, SQLite, HTML, CSS, JavaScript, and Bootstrap.

## Features

* Employee Login
* HR Login
* Apply Leave
* Leave History
* HR Approval/Rejection
* Dashboard Statistics
* Logout Functionality
* REST API Integration
* SQLite Database

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

### Backend

* C#
* ASP.NET Core Web API

### Database

* SQLite
* Entity Framework Core

## How to Run

### Backend

```bash
cd backend
dotnet run
```

### Frontend

Open:

```text
frontend/index.html
```

using Live Server.
